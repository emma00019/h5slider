# h5slider
-使用h5和css实现手机端左右切换
-克隆项目
```sh
    git clone https://github.com/emma00019/h5slider.git
```
-配置环境
```sh
    要使用grunt，第一件事是安装Node.js。（没用过也没关系，只需要安装之后让grunt能运行即可）。
    安装了node.js之后，用命令行工具执行一下命令：
        npm install -g grunt-cli
    确认grunt是否正确安装，可以使用以下命令进行查看：
        grunt --version
    然后在命令行中使用grunt命令就可以进行文件的压缩
```
-注释
```sh
    如果项目中没有package.json和gruntfile.js这两个文件，则需要创建。
    package.json文件让我们指定我们的开发环境所依赖的必须模块。有了它，项目的所有开发者都能保证安装上一直的必须模块，从而保证所有人拥有一样的开发环境。
    然后在命令行中运行npm install，这个命令告诉npm需要安装的必须模块，npm会安装它们，自动保存在项目根目录下一个叫做node_modules的文件夹里。
    有了gruntfile.js文件我们就可以使用grunt进行压缩
```
